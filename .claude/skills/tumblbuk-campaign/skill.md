# Tumblbuk Campaign — 크라우드펀딩 캠페인 제작

텀블벅 크라우드펀딩 캠페인을 기획하고 실행 가능한 수준의 산출물을 제작한다.

## 왜 텀블벅인가

- 한국 최대 크라우드펀딩 플랫폼으로, 창작/기술 프로젝트에 강한 사용자 베이스
- 스토리텔링 중심의 플랫폼 특성이 "가족 노래" 서비스의 감성과 잘 맞는다
- 펀딩 = 사전 판매 + 마케팅 + 시장 검증을 동시에 달성

## 캠페인 페이지 구조

### 1. 오프닝 (첫 스크롤)
- 핵심 한 줄: 감성적 후크 + 서비스 본질
- 예시: "엄마의 목소리로 부른 생일 축하 노래, 세상에 하나뿐인 선물"
- 히어로 이미지 또는 데모 영상 (30초)

### 2. 문제 제기 (Why)
- "특별한 날, 정말 특별한 선물을 하고 싶었던 적 있으신가요?"
- 기존 선물의 한계 → 목소리로 만든 노래의 차별점

### 3. 서비스 소개 (How)
- **3단계로 단순화:**
  1. 가족 목소리 녹음 (30초면 충분)
  2. 원하는 노래 스타일 선택
  3. AI가 만든 세상에 하나뿐인 노래 수령
- GIF 또는 짧은 영상으로 시각화

### 4. 샘플 결과물
- 실제 생성된 노래 2~3개 임베드 (다양한 장르)
- 전후 비교: 원본 목소리 → 생성된 노래

### 5. 리워드 구성

| 등급 | 가격 | 내용 |
|------|------|------|
| 얼리버드 | 19,900원 | 보이스 1개 + 오리지널 노래 1곡 |
| 기본 | 29,900원 | 보이스 1개 + 오리지널 노래 2곡 |
| 패밀리 | 49,900원 | 보이스 3개 + 노래 3곡 + 커버 1곡 |
| 프리미엄 | 79,900원 | 보이스 5개 + 노래 5곡 + 커버 3곡 + 고음질 WAV |

> 가격은 원가(API 비용) + 마진을 고려하여 ai-engineer와 협의 후 확정한다

### 6. 팀 소개
- 팀의 진정성과 기술력을 동시에 어필
- "왜 이 서비스를 만들게 되었는지" 개인 스토리

### 7. 일정
- 펀딩 기간: 30일
- 서비스 제공 시작: 펀딩 종료 후 2~4주

## 카피라이팅 톤

- **감성적이되 과장하지 않는다** — "AI가 완벽한 노래를 만든다" (X) → "가족의 목소리가 노래가 된다" (O)
- **기술보다 가치 중심** — Lyria, AI 같은 기술 용어는 최소화
- **행동 유도 명확** — 각 섹션 끝에 CTA (Call to Action)
- **한국어 자연스러움** — 번역체 금지, 구어체 허용

## 펀딩 목표 설정

- **최소 목표:** 200만원 (서버/API 비용 3개월치)
- **스트레치 목표:** 500만원 → 커버송 기능 추가, 1000만원 → 앱 개발
- 목표가 너무 높으면 달성 실패 리스크, 너무 낮으면 신뢰도 하락
