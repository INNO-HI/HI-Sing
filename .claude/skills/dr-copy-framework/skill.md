# DR Copy Framework

HI-Sing 랜딩페이지 전용 직접반응 카피라이팅 프레임워크.

## AIDA 프레임워크 (HI-Sing 적용)

### Attention - 주의 끌기
- "쑥스러움" 공감 후크: "마음은 가득한데, 전하기가 참 어렵습니다"
- 질문형 오프닝: "올해도 상품권을 드리려니 마음이 걸리시나요?"

### Interest - 관심 유지
- 기존 선물 한계 vs 노래 선물 차별성
- Before/After 대비: 매년 같은 선물 → 세상에 하나뿐인 노래

### Desire - 욕구 자극
- 실제 후기: "온 가족이 울었어요"
- 가격 앵커링: 4,000,000원 → 29,000원 (93% 절감)

### Action - 행동 유도
- CTA 문구 패턴: 행동 동사 + 혜택
- 긴급성: "한정 100명 얼리버드"

## PAS 프레임워크

- Problem: "올해도 상품권?"
- Agitation: "30년 치 감사를 건강하세요 한마디에"
- Solution: "노래로 전하면 어떨까?"

## CTA 문구 작성 규칙
- 행동 동사 + 혜택: "노래 선물 만들기" > "신청하기"
- 긴급성: "지금 만들면 부모님 생신에 딱 맞춰"
- 3안 이상 제시: A/B 테스트용

## 한국어 DR 카피 패턴
- 숫자의 힘: "93% 비용 절감", "5~7일", "30초면 충분"
- 한정성: "한정 100명", "얼리버드 26% 할인"
- 감성 후크: "~하신 적 있으신가요?", "~이 참 어렵습니다"
- 금지: AI 강조, 기술 자랑, 번역체
