# DR Conversion Playbook

HI-Sing 랜딩페이지 전환율 최적화 전략 플레이북.

## 전환 심리학 7대 원칙

### 1. 가격 앵커링
- 기존: 4,000,000원 (전문 작곡) → HI-Sing: 29,000원
- 시각적 대비: 다크 배경 큰 숫자 vs 밝은 배경 강조 숫자
- "93% 비용 절감" 명시

### 2. 사회적 증거
- 고객 후기 슬라이더 (LP디스크 UI)
- 숫자 증거 추가: "N명 주문 완료", "만족도 98%"
- 인용문 강조: "온 가족이 울었어요"

### 3. 긴급성
- "한정 100명 얼리버드 26% 할인"
- "부모님 생신 전 주문 마감"
- 카운트다운 타이머 (선택)

### 4. 희소성
- "한정 수량", "이번 달만"
- 얼리버드 잔여 수량 표시

### 5. 리스크 리버설
- "1회 무료 수정"
- "데이터 30일 후 삭제"
- "마음에 안 들면 수정"

### 6. 호혜성
- 무료 샘플 듣기 → 주문 욕구 자극

### 7. 일관성
- 샘플 듣기(작은 동의) → 가격 확인 → 구매(큰 동의)

## 스크롤 깊이별 전략

| 깊이 | 목표 | 요소 |
|------|------|------|
| 0~25% | 공감 + 호기심 | 히어로, 공감, 채팅버블 |
| 25~50% | 해결책 + 증거 | Before/After, 프로세스, AI 설명 |
| 50~75% | 가격 + 비교 | 가격 앵커링, 포함 내역, 특별함 |
| 75~100% | 후기 + 행동 | 체크리스트, 후기 슬라이더, CTA |

## A/B 테스트 가설 템플릿
- 가설: [변경 사항]을 적용하면 [지표]가 [방향]할 것이다
- 변수: CTA 문구, 색상, 위치, 가격 제시, 후기 형태
- 측정: 클릭률, 스크롤 깊이, 전환율
