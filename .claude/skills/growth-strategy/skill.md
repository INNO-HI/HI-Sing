# Growth Strategy — HI-Sing 마케팅 전략 수립

HI-Sing의 서비스 특성에 맞는 마케팅 전략을 수립한다.

## 이 서비스가 일반 서비스와 다른 점

| 일반 SaaS/커머스 | HI-Sing |
|-----------------|---------|
| 반복 구매/구독 | **이벤트 있을 때만** 1회성 구매 |
| 기능/가격으로 비교 구매 | **감정**으로 구매 ("감동이겠다") |
| 광고로 직접 전환 | **입소문/바이럴**이 핵심 성장 동력 |
| 연중 균일한 수요 | **시즌 집중** (생신, 기념일, 명절) |
| 기술력/성능 어필 | **편의성** 어필 ("귀찮지 않게 특별한 선물") |

이 차이점 때문에 일반적인 마케팅 프레임워크를 그대로 적용하면 안 된다.

## 전략 수립 프레임워크

### 1. 바이럴 루프 설계

HI-Sing의 최대 자산은 **선물을 받는 사람이 다음 고객**이 되는 구조다.

```
주문자 → 노래 제작 → 선물 전달 → 받는 사람 감동
                                    ↓
                              카톡/SNS 공유 ← 여기를 극대화
                                    ↓
                              공유 본 사람
                                    ↓
                           "나도 해볼까?" (다음 이벤트에)
```

**바이럴 계수(K-factor) 높이는 방법:**
- 선물 수신 페이지에 자연스러운 서비스 소개 (강제적이지 않게)
- 공유하기 쉬운 형태로 결과물 제공 (카톡 링크, 인스타 스토리 사이즈)
- "나도 만들어보기" 버튼을 수신 페이지에 배치
- 받은 사람에게 할인 쿠폰 제공 (다음 주문 유도)

**K-factor 목표:**
- K > 1이면 자연 성장 (1명이 1명 이상을 데려옴)
- 현실적 초기 목표: K = 0.3~0.5 (광고로 보충)
- 장기 목표: K > 0.7 (광고 의존도 최소화)

### 2. 시즌 마케팅 캘린더

HI-Sing 수요는 특정 시즌에 집중된다. 시즌 2~3주 전부터 마케팅을 시작하되, 제작 기간(5~7영업일)을 고려해 **이벤트 2주 전까지 주문 유도**한다.

| 시기 | 이벤트 | 마케팅 강도 | 메시지 |
|------|--------|-----------|--------|
| 1~2월 | 설 명절 | ★★☆ | "올 설엔 용돈 대신 노래 한 곡" |
| 3월 | 화이트데이, 졸업 | ★☆☆ | 보조 시즌 |
| 4~5월 | **어버이날, 부모님 생신** | ★★★ | **최대 시즌** — 예산 집중 |
| 5월 | 가정의 달 | ★★★ | 어버이날 연계 |
| 6~8월 | 비시즌 | ★☆☆ | 결혼기념일 타겟팅 |
| 9~10월 | **추석, 칠순/팔순** | ★★★ | 두번째 최대 시즌 |
| 11월 | 빼빼로데이 | ★☆☆ | 보조 시즌 |
| 12월 | 크리스마스, 연말 | ★★☆ | "올 한 해 감사한 사람에게" |

**예산 배분 원칙:** 연간 예산의 60%를 4~5월 + 9~10월에 집중

### 3. 채널 전략

| 채널 | 역할 | 메시지 유형 | 타겟 | 예상 CAC |
|------|------|-----------|------|---------|
| **인스타그램 릴스** | 인지 + 감정 유발 | 감동 반응 영상, 비포/애프터 | 30~50대 여성 | 5,000~8,000원 |
| **유튜브 숏츠** | 인지 + 설명 | "이런 선물 어때요?" 시리즈 | 30~50대 전체 | 3,000~6,000원 |
| **카카오톡 채널** | 리마인드 + 전환 | 시즌 알림, 할인 쿠폰 | 기존 관심자 | 낮음 (오가닉) |
| **네이버 블로그/카페** | 검색 유입 | 상세 후기, 체험기 | 검색 의도 있는 사람 | 2,000~4,000원 |
| **바이럴 (수신 페이지)** | 자연 전환 | "나도 만들어보기" | 선물 받은 사람 | 0원 |

**초기 전략 (예산 제한):**
1. 바이럴 루프 최적화 (비용 0)
2. 인스타그램 릴스 소규모 테스트 (10~30만원/월)
3. 효과 검증 후 유튜브/네이버 확장

### 4. 콘텐츠 전략

**감정 유발 콘텐츠 (인지 단계):**
- "칠순잔치에서 아들이 준비한 깜짝 노래" (반응 영상 형태)
- "엄마가 아들 목소리로 만든 노래 듣고 운 이유" (스토리텔링)
- 비포/애프터: 일상 음성 → 완성 노래 (변환의 놀라움)

**정보 콘텐츠 (관심 단계):**
- "3분이면 끝나는 세상에서 가장 특별한 선물"
- "29,000원으로 부모님 울리는 법"

**시즌 콘텐츠 (전환 단계):**
- "어버이날 선물 아직 안 정했다면"
- "이번 추석엔 다르게"

**핵심 원칙:**
- 기술 설명 금지. "AI가 목소리를 학습하여..." (X)
- 결과/감정 중심. "엄마가 눈물 흘렸어요" (O)
- 귀찮음 해소 강조. "3분이면 끝" (O)

### 5. 퍼널 최적화

각 단계별 이탈 원인과 대응:

| 퍼널 단계 | 이탈 원인 | 대응 |
|----------|----------|------|
| 인지→관심 | "뭔지 모르겠다" | 샘플 노래 즉시 재생 |
| 관심→결제 | "비싼 거 아닌가" | 가격 명확 표시, 얼리버드 할인 |
| 관심→결제 | "품질이 괜찮을까" | 비포/애프터 비교, 후기 |
| 결제→제작요청 | "귀찮다, 나중에" | 이메일 리마인드, "3분이면 끝" 강조 |
| 선물→공유 | "공유까지는 안 하지" | 공유하기 쉬운 UI, 카카오톡 원클릭 |

### 6. KPI 정의

| 지표 | 정의 | 초기 목표 |
|------|------|----------|
| **K-factor** | 1주문당 유입된 신규 방문자 수 | 0.3+ |
| **전환율** | 랜딩 방문 → 결제 | 3~5% |
| **CAC** | 유료 채널 기준 고객 획득 비용 | < 10,000원 |
| **공유율** | 노래 수신 → 카톡/SNS 공유 | 30%+ |
| **시즌 매출 비중** | 피크 시즌 매출 / 전체 매출 | 60%+ |
| **재주문율** | 1회 주문자의 12개월 내 재주문 | 15%+ |

## 산출물 형식

마케팅 전략서에는 다음을 포함한다:
1. 핵심 인사이트 (시장/고객/경쟁)
2. 바이럴 루프 설계도
3. 채널 전략 + 예산 배분
4. 시즌 마케팅 캘린더
5. 콘텐츠 유형별 기획
6. KPI + 측정 방법
7. 초기 3개월 실행 로드맵
