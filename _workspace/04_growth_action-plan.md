## Part 1: 마케팅 지표 추적 세팅 (런칭 전 필수)

런칭 전에 아래 세팅을 모두 완료해야 한다. 데이터 없이 운영하면 어디서 유입이 오는지, 뭐가 되고 있는지 전혀 알 수 없다.

### 1-1. Google Analytics 4 (GA4) 세팅

**자체 스토어 + 글로벌 웹에 적용** (텀블벅은 자체 통계 사용)

**계정 생성 및 설치 (30분 소요)**

1. [analytics.google.com](https://analytics.google.com)에 접속하여 Google 계정으로 로그인
2. 좌측 하단 톱니바퀴(관리) 클릭 → "계정 만들기" 클릭
3. 계정 이름: "HI-Sing" 입력 → 다음
4. 속성 이름: "HI-Sing Store" 입력, 보고 시간대: "대한민국", 통화: "한국 원(KRW)" → 다음
5. 비즈니스 정보: 업종 "예술/엔터테인먼트", 규모 "소규모" 선택 → 만들기
6. 데이터 스트림 추가: "웹" 선택 → 자체 스토어 URL 입력 → 스트림 만들기
7. "측정 ID" (G-XXXXXXXXXX 형태)를 복사하여 웹사이트 `<head>` 태그에 GA4 스크립트 삽입
8. 글로벌 웹도 동일하게 별도 데이터 스트림 추가 (같은 속성 내에서)

**이벤트 추적 설정 (1시간 소요)**

GA4 관리 화면 → 데이터 표시 → 이벤트에서 아래 맞춤 이벤트를 생성한다.

| 이벤트 이름 | 트리거 시점 | 용도 |
|------------|-----------|------|
| `sample_play` | 샘플 노래 재생 버튼 클릭 시 | 관심도 측정 |
| `begin_order` | 주문하기/결제하기 버튼 클릭 시 | 구매 의향 측정 |
| `purchase_complete` | 결제 완료 시 | 매출 측정 |
| `share_click` | 공유 버튼 클릭 시 | 바이럴 측정 |
| `gift_page_view` | 선물 수신 페이지 조회 시 | 선물 도달 측정 |
| `gift_to_order` | 선물 페이지에서 "나도 선물하기" 클릭 시 | 바이럴 전환 측정 |

개발팀에게 위 이벤트를 해당 버튼/페이지에 `gtag('event', '이벤트명')` 코드로 심어달라고 요청한다.

**전환 설정**

1. GA4 관리 → 데이터 표시 → 이벤트 → `purchase_complete` 옆 토글을 켜서 "전환으로 표시"
2. `gift_to_order`도 동일하게 전환으로 표시
3. 이렇게 하면 GA4 보고서에서 전환 수, 전환율이 자동 집계됨

### 1-2. UTM 파라미터 체계

모든 외부 링크에 UTM을 붙여야 유입 채널별 성과를 구분할 수 있다.

**UTM 구조 (복사해서 사용)**

```
https://hi-sing.com/?utm_source={소스}&utm_medium={매체}&utm_campaign={캠페인}&utm_content={콘텐츠}
```

**채널별 UTM 표준 (이 표를 팀 공용 문서에 붙여놓을 것)**

| 채널 | utm_source | utm_medium | utm_campaign | 예시 |
|------|-----------|-----------|-------------|------|
| 네이버 블로그 | naver_blog | organic | launch_phase1 | `?utm_source=naver_blog&utm_medium=organic&utm_campaign=launch_phase1` |
| 맘카페 댓글 | momcafe | community | launch_phase1 | `?utm_source=momcafe&utm_medium=community&utm_campaign=launch_phase1` |
| 클리앙 | clien | community | launch_phase1 | `?utm_source=clien&utm_medium=community&utm_campaign=launch_phase1` |
| 인스타 광고 | instagram | paid_social | launch_phase2 | `?utm_source=instagram&utm_medium=paid_social&utm_campaign=launch_phase2` |
| 페이스북 광고 | facebook | paid_social | launch_phase2 | `?utm_source=facebook&utm_medium=paid_social&utm_campaign=launch_phase2` |
| 카카오톡 공유 | kakao | share | viral | `?utm_source=kakao&utm_medium=share&utm_campaign=viral` |
| 선물 페이지 | gift_page | referral | viral | `?utm_source=gift_page&utm_medium=referral&utm_campaign=viral` |

**UTM 링크 생성 방법**

1. [ga-dev-tools.google/ga4/campaign-url-builder](https://ga-dev-tools.google/ga4/campaign-url-builder/) 접속
2. 위 표에 맞춰 각 필드 입력
3. 생성된 URL을 복사하여 사용
4. 커뮤니티에 직접 링크를 넣지 않으므로(원칙상 "검색" 유도), UTM은 주로 블로그 글, SNS 광고, 카카오 공유 링크에 사용

> **주의:** 커뮤니티 시딩 글/댓글에는 링크를 넣지 않는다. "하이싱 검색해보세요"로 유도. UTM은 블로그, 광고, 공유 링크 등 직접 URL을 삽입하는 채널에만 사용.

### 1-4. 텀블벅 자체 통계 활용

텀블벅은 외부 트래킹을 심을 수 없으므로, 텀블벅 크리에이터 대시보드를 활용한다.

- 텀블벅 크리에이터 센터 → 내 프로젝트 → 통계에서 확인 가능한 항목:
  - 일일 서포터 수, 누적 달성금액, 달성률
  - 유입 경로 (텀블벅 내부 vs 외부)
  - 리워드별 선택 비율
- 매일 오전 10시에 서포터 수, 달성금액을 스프레드시트에 기입

- 이미지 5장 이상: 서비스 화면 캡처 2장, 실제 노래 재생 화면 1장, 가족 관련 감성 이미지 2장

### 3-1. 역할 배분

### 4-1. 에이전시에게 전달할 브리프

아래 내용을 문서로 정리하여 에이전시에게 전달한다.

---

**HI-Sing 마케팅 에이전시 브리프**

**서비스 소개**
- HI-Sing: 가족의 목소리를 녹음하여 AI로 오리지널 노래 또는 커버곡을 만들어주는 선물 서비스
- 주요 고객: 30~50대, 부모님/가족에게 특별한 선물을 하고 싶은 사람
- 가격대: 3만원~7만원대
- 판매 채널: 텀블벅 (크라우드펀딩), 자체 스토어 (국내), 글로벌 웹사이트

**월 예산: 90만원**

**예산 배분 안**

| 항목 | 금액 | 비율 | 설명 |
|------|------|------|------|
| 인스타그램 릴스 광고 운영 | 30만원 | 33% | 타겟팅, 소재 관리, 최적화 |
| 페이스북 광고 운영 | 15만원 | 17% | 40~50대 타겟, 사연형 콘텐츠 |
| 콘텐츠 제작 (릴스/숏츠) | 20만원 | 22% | 월 4~6개 영상 소재 제작 |
| 커뮤니티 관리 (시딩 유지) | 15만원 | 17% | 주 2~3회 댓글, 월 2~3개 글 |
| 에이전시 운영 수수료 | 10만원 | 11% | 리포팅, 커뮤니케이션 |

> 참고: 위 배분은 제안이며, 에이전시와 협의하여 조정 가능. 핵심은 "광고비 45만원 + 콘텐츠 제작 20만원 + 커뮤니티 15만원 + 수수료 10만원" 구조.

**에이전시에게 요청할 업무**

1. **인스타그램/페이스북 광고 운영**
   - 월 소재 4~6개 제작 (15~30초 릴스/카드뉴스)
   - 타겟 설정: 30~50대, 관심사 "가족/선물/부모님"
   - A/B 테스트: 소재 2~3개 동시 집행, 반응 좋은 소재에 예산 집중
   - 주간 성과 리포트 제출
2. **커뮤니티 시딩 유지**
   - 주 2~3회 기존 "선물 추천" 글에 댓글 (자연스러운 톤, 우리 시딩 가이드라인 전달)
   - 월 2~3개 커뮤니티 글 (체험기, 정보 공유)
   - 글/댓글 사전 초안을 팀에 공유하여 승인 후 게시
3. **월간 리포팅**
   - 광고 성과: 노출, 클릭, CTR, CPC, 전환 수
   - 커뮤니티 성과: 글/댓글 수, 조회수, 자연 반응
   - 개선 제안

**전달할 자료**
- 서비스 소개 자료 (스크린샷, 샘플 노래 링크)
- 커뮤니티 시딩 가이드라인 (본 문서 Part 3-4 섹션)
- 기존 시딩 글/댓글 목록 (1~2주차 활동 내역)
- GA4 보기 권한 (읽기 전용)
- 텀블벅 대시보드 스크린샷 (매주 공유)

> **팀이 반드시 계속할 일:** 블로그 SEO 글 (월 2~3개), 고객 응대, 주문/제작 관리, 전략 의사결정. 이것들은 에이전시에게 맡기면 안 된다.

### 4-3. 월 90만원으로 할 수 있는 현실적 범위

**할 수 있는 것:**
- 인스타/페이스북 광고: 일 1.5만원 수준 집행 (월 광고비 45만원). 월 3만~5만 노출, 500~1,000 클릭 예상
- 월 4~6개 광고 소재 제작 (템플릿 기반 간단 영상/이미지)
- 커뮤니티 댓글 주 2~3회, 글 월 2~3개
- 기본 리포팅 (주간 or 격주)

**할 수 없는 것 (추가 예산 필요):**
- 인플루언서 협업 (1건당 20~50만원 별도)
- 고품질 영상 제작 (전문 촬영/편집)
- 유튜브 채널 운영 (별도 채널 매니저 필요)
- 대규모 광고 캠페인 (월 300만원+ 필요)

### 4-4. 에이전시 성과 측정 기준

매월 아래 기준으로 에이전시 성과를 평가한다.

| 지표 | 목표 | 미달 시 대응 |
|------|------|-----------|
| SNS 광고 CTR (클릭률) | 2% 이상 | 소재 변경 요청. 2개월 연속 미달 시 에이전시 교체 검토 |
| SNS 광고 CPC (클릭당 비용) | 500원 이하 | 타겟팅 재조정 요청 |
| 광고 → 결제 전환율 | 3% 이상 | 랜딩페이지 개선 or 소재 변경 |
| 커뮤니티 글 삭제율 | 20% 이하 | 시딩 가이드라인 재교육 |
| 월간 리포트 제출 | 매월 1일 | 미제출 시 경고, 2회 시 계약 해지 |
| 전체 CAC (고객 획득 비용) | 10,000원 이하 | 채널 배분 재조정 |

### 4-5. 에이전시 선정 기준 / 체크리스트

90만원 예산에 맞는 에이전시는 대형 에이전시가 아니라 소규모 마케팅 대행사 또는 프리랜서 마케터이다.

**에이전시 찾는 방법:**
1. **크몽** (kmong.com) → "SNS 마케팅 대행" 검색 → 월 정기 운영 가능한 전문가 필터
2. **숨고** (soomgo.com) → "SNS 광고 대행" → 견적 요청
3. **위시켓** (wishket.com) → 마케팅 프로젝트 등록
4. **인스타그램** → "마케팅대행" 해시태그 → 포트폴리오 확인

**선정 체크리스트 (후보 3곳 이상 비교):**

| 항목 | 확인 질문 | 합격 기준 |
|------|----------|----------|
| 유사 경험 | "소비재/서비스 마케팅 경험이 있는가?" | B2C 서비스 운영 경험 2건 이상 |
| 포트폴리오 | "실제 운영한 광고 성과를 보여줄 수 있는가?" | CTR 2% 이상 달성 사례 보유 |
| 커뮤니티 시딩 경험 | "커뮤니티 시딩/바이럴 마케팅 경험이 있는가?" | 맘카페, 네이버 카페 시딩 경험 우대 |
| 리포팅 방식 | "주간/월간 리포트를 어떤 형태로 제공하는가?" | 구글 스프레드시트 or 노션으로 실시간 공유 |
| 소통 방식 | "평소 소통은 어떻게 하는가?" | 슬랙 or 카카오톡 당일 응답 가능 |
| 최소 계약 기간 | "최소 몇 개월 계약인가?" | 1~2개월 단기 계약 가능 (성과 확인 후 연장) |
| 광고 계정 소유권 | "광고 계정은 누구 명의로 만드는가?" | 반드시 HI-Sing 명의. 에이전시 명의 X |
| 해지 조건 | "중간에 해지하면 위약금이 있는가?" | 위약금 없는 조건 or 1개월 전 통보 |

**계약 전 반드시 확인:**
- 1개월 시범 운영 후 성과에 따라 계속 여부 결정하는 조건 삽입
- 광고 계정, GA4 접근 권한은 HI-Sing 소유로 유지
- 커뮤니티 시딩 글은 사전 승인 후 게시 (무단 게시 금지)
