# HI-Sing 마케팅 전략서

> **노래로 전하는 우리 가족 이야기**
> 런칭 전 씨뿌리기 → 검색 기반 신뢰 확보 → 바이럴 확대

---

## 1. 전략 개요

### 핵심 인사이트

**모방 리스크가 가장 큰 위협이다.** AI 시대에 기술적 진입장벽이 낮기 때문에, 사전 홍보 기간이 길수록 모방자에게 준비 시간을 주는 꼴이다. 따라서:

- 사전 기대감 조성 기간을 두지 않는다
- **3채널 동시 런칭 + 씨뿌리기 동시 진행**으로 속도전을 건다
- 런칭 즉시 결제/제작이 가능해야 한다 (텀블벅 사전알림 없이 바로 오픈)

동시에, 타겟 고객(30~50대)은 검색으로 신뢰를 확인하는 행동 패턴이 있다. 씨뿌리기를 런칭과 동시에 시작하되, **첫 1~2주는 SNS 광고 없이 커뮤니티 시딩에 집중**하여 검색 기반을 확보한 후 바이럴을 확대한다.

### 런칭 채널 (3채널 동시 오픈)

| 채널 | 역할 | 결제 |
|------|------|------|
| **텀블벅** | 국내 크라우드펀딩 (초기 모객) | 텀블벅 내 결제 |
| **자체 스토어** | 국내 상시 판매 | 토스페이먼츠 등 자체 PG |
| **글로벌 웹사이트** | 해외 시장 | Stripe 결제 |

> 3채널 모두 오픈 당일부터 결제 → 제작 가능

### 전체 로드맵

| Phase | 기간 | 핵심 활동 | 목표 |
|-------|------|----------|------|
| **Phase 0: 사전 준비** | 런칭 1주 전 | 블로그 SEO 글 5~7개 발행, 커뮤니티 계정 활동 시작 | 검색 인덱싱 + 계정 신뢰도 |
| **Phase 1: 런칭 + 씨뿌리기** | D-Day ~ 2주 | **3채널 동시 오픈** + 커뮤니티 댓글·글 + 블로그 체험기 | 첫 결제 유입 + "하이싱" 검색 시 글 10개+ |
| **Phase 2: 바이럴 확대** | 3주~6주 | SNS 광고, 인플루언서 협업, 릴스/숏츠 | 텀블벅 달성률 300% + 자체 스토어 매출 |
| **Phase 3: 시즌 마케팅** | 7주~ | 시즌별 집중 마케팅, 바이럴 루프 강화 | 안정적 반복 매출 |

## 2. Phase 1: 런칭 + 씨뿌리기 동시 진행 (D-Day ~ 2주)

> **핵심:** 사전 홍보 없이 3채널 동시 오픈. 런칭과 동시에 씨뿌리기 시작.
> 모방 리스크 때문에 "곧 나옵니다" 기간을 두지 않는다.

### 2-2. 글/댓글 유형별 예시 (실전용)

#### 시나리오 1: "부모님 선물 추천" 글에 댓글

> **대상:** 맘카페, 블라인드, 82cook에서 "시아버지 칠순 선물 뭐가 좋을까요?" 류의 글

**댓글 예시 A (짧은 추천)**
```
저는 올해 아버지 칠순때 노래 만들어서 선물했어요.
하이싱이라는 곳인데, 제 목소리 녹음해서 보내면
아버지한테 드릴 노래를 만들어주더라고요.
틀어드렸더니 아버지가 눈물을 훔치시더라구요ㅠㅠ
가격도 생각보다 부담 없어서 추천드려요.
```

**댓글 예시 B (비교형)**
```
건강식품이나 상품권 말고 좀 특별한 거 드리고 싶으시면
노래 선물은 어떠세요? 저도 처음엔 반신반의했는데
실제로 받아보니 퀄리티가 좋더라고요.
하이싱이라고 검색하면 나와요.
목소리 녹음 보내면 그걸로 노래를 만들어줘요.
어머니가 두고두고 들으신대요 ㅎㅎ
```

**댓글 예시 C (구체적 상황)**
```
저도 매년 같은 고민이었는데 올해는 좀 달랐어요.
남편이 시어머니한테 드릴 선물 고민하길래
노래 선물 서비스 추천해줬거든요.
남편 목소리로 감사 노래를 만들어서 드렸는데
시어머니가 울면서 좋아하셨어요.
하이싱으로 검색해보세요. 텀블벅에서 펀딩 중이에요.
```

하이싱이라고 하는데, 제 목소리를 녹음해서 보내면

처음엔 솔직히 "이게 되나?" 싶었는데
완성본 받아보고 소름 돋았어요.
제 목소리인데 노래를 부르고 있는 거예요.

### 2-3. 주의사항

| 항목 | 지침 |
|------|------|
| **톤** | 반말+존댓말 혼용, 각 커뮤니티 분위기에 맞추기. 맘카페는 "~요" 체, 클리앙/뽐뿌는 "~다" 체 |
| **광고 티** | 절대 서비스명을 제목에 넣지 않는다. 본문 중간~하단에 자연스럽게 언급 |
| **링크** | 직접 링크 금지. "검색하면 나와요"로 유도. 링크 넣으면 광고로 신고됨 |
| **계정 관리** | 새 계정으로 바로 글 쓰지 않는다. 최소 1주일 전부터 일반 댓글 5~10개 달아놓기 |
| **빈도** | 하루 1~2개 이상 같은 커뮤니티에 쓰지 않는다. 패턴이 보이면 역효과 |
| **반응 관리** | 질문이 달리면 반드시 답글. "어디서 해요?" → "하이싱 검색해보세요" |
| **사진** | 가능하면 실제 재생 화면 캡처 or 가족사진(모자이크) 첨부. 리얼감 상승 |
| **다양성** | 같은 문구 복사붙여넣기 금지. 상황별로 다르게 작성. 가족 구성, 이벤트, 감정 다 다르게 |

### 2-4. 네이버 블로그 SEO (검색 대비)

검색 시 나와야 할 글을 **런칭 전~1주차**에 미리 발행한다.

**타겟 키워드:**

| 키워드 | 월 검색량(추정) | 글 유형 |
|--------|--------------|---------|
| 부모님 생신 선물 | 높음 | 리스트형 ("부모님 생신 선물 추천 BEST 5") |
| 부모님 칠순 선물 | 중간 | 체험기형 ("칠순 선물로 노래 선물한 후기") |
| 어버이날 선물 추천 | 높음(시즌) | 리스트형 + 체험기 |
| 특별한 가족 선물 | 중간 | 정보형 ("2026년 특별한 가족 선물 아이디어") |
| 하이싱 | 없음→만들기 | 서비스 소개 + 후기 |
| 노래 선물 | 낮음 | 체험기 + 서비스 비교 |

**블로그 글 발행 계획 (최소 5개):**

1. **"부모님 생신 선물 추천 BEST 5 (2026년)"** — HI-Sing을 5개 중 하나로 자연스럽게 소개
2. **"아버지 칠순 선물로 노래를 만들어봤습니다 (하이싱 후기)"** — 체험기 형태
3. **"어버이날 선물 고민? 올해는 이렇게 해보세요"** — 선물 아이디어 중 하나로 배치
4. **"가족 목소리로 노래 만드는 하이싱, 직접 써봤어요"** — 브랜드 키워드 검색용
5. **"부모님한테 상품권 말고 감동 선물 드리는 법"** — 감성 글 + HI-Sing 언급

**블로그 SEO 요령:**
- 제목에 타겟 키워드 포함 (앞쪽 배치)
- 본문 1,500자 이상, 이미지 5장 이상
- 서비스 캡처 화면, 결과물 플레이어 화면 포함
- 첫 문단에서 공감 → 중간에 서비스 소개 → 후반에 사용법/가격 → 마무리 감상
- 발행 후 서로이웃 추가, 공감 요청으로 노출 부스트

## 3. Phase 2: 바이럴 마케팅 (3주차~)

### 3-1. SNS 채널별 전략

#### 인스타그램 릴스 (메인 채널)

**왜 메인인가:** 30~50대 여성 이용률 가장 높음. 감성 콘텐츠 바이럴 최적.

| 콘텐츠 유형 | 형식 | 예시 |
|------------|------|------|
| **리액션 영상** | 15~30초 릴스 | 부모님이 노래 듣고 감동하는 순간 (실제 or 연출) |
| **비포/애프터** | 15초 릴스 | 목소리 녹음 → 완성된 노래 비교 |
| **가격 비교** | 카드뉴스 | "상품권 5만원 vs 세상에 하나뿐인 노래 3만원" |
| **고객 사연** | 스토리 | 실제 주문자의 사연 소개 (동의 하에) |

**광고 세팅:**
- 타겟: 30~50대, 관심사 "가족", "선물", "부모님"
- 일 예산: 1~2만원 (월 30~50만원)
- CTA: "지금 펀딩 참여하기" → 텀블벅 링크
- A/B 테스트: 리액션 영상 vs 설명형, 2~3개 소재 동시 집행

#### 유튜브 숏츠

**왜 필요한가:** 검색 기반 유입 + 숏츠 추천 알고리즘.

| 콘텐츠 유형 | 형식 | 예시 |
|------------|------|------|
| **감동 숏츠** | 30~60초 | "아버지 칠순에 노래 선물했더니..." (결과 리액션) |
| **과정 소개** | 30초 | "목소리 녹음 → 노래 완성" 타임랩스 느낌 |
| **비교** | 15초 | "매년 같은 선물 vs 올해는 다르게" |

- 광고 집행보다는 오가닉 콘텐츠 위주
- 주 2~3개 업로드
- 댓글에 텀블벅/스토어 링크

#### 페이스북

**왜 필요한가:** 40~50대 도달률 최고. 공유 기능으로 바이럴.

| 콘텐츠 유형 | 형식 | 예시 |
|------------|------|------|
| **사연 카드** | 이미지+텍스트 | "작년 아버지 팔순, 드릴 게 없어 막막했는데..." |
| **페이스북 광고** | 영상 15초 | 리액션 영상 + "지금 펀딩 참여" CTA |

- 타겟: 35~55세, 자녀 있는 기혼자
- 일 예산: 5천원~1만원 (월 15~30만원)

#### 카카오톡 채널

- 자체 카카오 채널 개설
- 주문 진행 상황 알림 발송용
- 시즌 전 리마인드 메시지 (어버이날 2주 전 등)

### 3-2. 콘텐츠 핵심 원칙

1. **"감동 리액션"이 가장 강력한 콘텐츠다.** 노래를 듣고 눈물짓는 부모님 영상 하나가 설명 100개보다 낫다.
2. **기술 설명 최소화.** "목소리로 노래를 만든다"까지만. 작동 원리는 안 알려줘도 된다.
3. **가격 허들 낮추기.** "커피 몇 잔 값으로 세상에 하나뿐인 선물"
4. **쑥스러움 대리 해소.** "직접 노래 못 부르니까, 대신 만들어드려요"

## 4. 바이럴 루프 설계

### 4-1. 핵심 루프

```
[주문자] 노래 주문 + 결제
    ↓
[HI-Sing] 노래 제작 → 완성
    ↓
[주문자] 완성 노래를 가족에게 선물 (공유 링크)
    ↓
[선물 받는 사람] 감동 → "이걸 어디서 만들었어?"
    ↓
[선물 받는 사람의 주변] "나도 부모님한테 해드리고 싶다"
    ↓
[신규 고객] 검색 → 주문
    ↓
(루프 반복)
```

### 4-2. K-factor 높이는 구체적 방법

**K-factor = 초대 수 x 전환율**

#### 공유를 유도하는 장치들

| 장치 | 상세 | 기대 효과 |
|------|------|----------|
| **카카오톡 선물 링크** | 완성 노래를 카카오톡으로 보낼 수 있는 전용 링크. 예쁜 카드 디자인 + 재생 버튼 | 받는 사람이 "이게 뭐야?" → 서비스 인지 |
| **공유 카드 자동 생성** | 노래 완성 시 "OO님의 가족에게 전하는 노래" 이미지 카드 자동 생성. 인스타 스토리 규격 | SNS 공유 허들 낮춤 |
| **"다음 선물" 할인 쿠폰** | 주문자에게 "다음 선물 20% 할인" + "친구에게 추천하면 10% 추가 할인" | 재구매 + 추천 동시 유도 |
| **가족 단체 할인** | "가족 3명이 각각 주문하면 전원 20% 할인" (형제자매가 함께 선물) | 한 가정에서 복수 주문 |
| **리뷰 인센티브** | "리뷰 작성하면 다음 주문 5,000원 할인" (네이버 블로그 리뷰 포함) | 자발적 후기 생산 → SEO 강화 |

#### 선물 수신 페이지 최적화

선물 받는 사람이 보는 페이지가 바이럴의 핵심이다.

```
┌──────────────────────────────┐
│                              │
│  OO님에게 전하는 노래         │
│                              │
│  [▶ 재생 버튼]               │
│  ♪ 노래 재생                 │
│                              │
│  ─────────────────────────   │
│                              │
│  "이 노래는 OO님의 가족이    │
│   마음을 담아 만든            │
│   세상에 하나뿐인 노래입니다" │
│                              │
│  [💌 카카오톡으로 답장하기]   │
│  [🎵 나도 노래 선물하기]     │  ← 핵심 CTA
│                              │
└──────────────────────────────┘
```

- "나도 노래 선물하기" 버튼을 눈에 띄게 배치
- 선물 받은 사람이 감동한 직후가 전환율 최고 타이밍
- 클릭하면 HI-Sing 스토어로 이동 + "첫 주문 10% 할인" 팝업

#### 바이럴 부스터

- **생일/기념일 리마인더:** 주문자에게 "작년 이맘때 OO님께 노래를 선물하셨어요. 올해도 특별한 선물을 준비해보세요" 알림
- **"우리 가족 노래 모음":** 여러 번 주문한 고객은 가족 노래 앨범 페이지 자동 생성 → 공유 가능
- **잔치/이벤트 현장 활용:** "칠순잔치에서 틀어보세요" → 참석자 전원이 서비스 인지 → 여러 가정으로 확산

| 월 주문 수 | 50건 이상 | 자체 대시보드 |
| 바이럴 계수 (K-factor) | 0.3 이상 | 추천 유입 / 총 주문 |
| 공유 링크 클릭률 | 30% 이상 (받는 사람 중) | 링크 트래킹 |
| "나도 선물하기" 전환율 | 5% 이상 (공유 링크 방문자 중) | GA4 이벤트 |
| 재구매율 | 20% 이상 (연간) | 자체 대시보드 |
| NPS (순추천지수) | 60 이상 | 설문 |
| 리뷰 작성률 | 주문자의 15% 이상 | 리뷰 수 / 주문 수 |

### 핵심 퍼널 지표 (상시 모니터링)

```
[인지] SNS 노출 / 검색 노출
  ↓  CTR 2%+
[방문] 랜딩페이지 방문
  ↓  이탈률 60% 이하
[관심] 샘플 노래 재생
  ↓  재생률 40%+
[결제] 텀블벅/스토어 결제
  ↓  전환율 5%+
[완료] 노래 수신
  ↓  만족도 4.5/5+
[공유] 가족에게 선물 + SNS 공유
  ↓  공유율 70%+
[재유입] 선물 받은 사람의 주변에서 재주문
  ↓  K-factor 0.3+
```
