## 3. 자체 사이트 이벤트 섹션 기획

### 🎯 목적
텀블벅 펀딩 진행 중, `hisolution.site` 방문자를 **텀블벅으로 유도 + 바이럴 루프 생성**

### 📍 위치
홈 히어로 **하단 바로 아래** (1번째 스크롤 영역) + 플로팅 배너 (우측 하단)

### 🎁 이벤트 구성

#### 이벤트 1 — 얼리버드 특가 배너 (메인)
- **헤드라인**: `🎉 어버이날 펀딩 진행 중 — 31% 특가`
- **서브**: "텀블벅에서만 19,900원 (원가 29,000원)"
- **카운트다운**: `⏰ 펀딩 마감까지 D-XX` (동적)
- **CTA 버튼**: `▶ 텀블벅 바로가기`
- **디자인**: primary-50 배경 + primary-400 강조

#### 이벤트 2 — 추천인 리워드 (바이럴 루프)
- **메커니즘**: 친구 초대 링크 공유 → 2명 펀딩 성공 시 추가 가사카드 액자 1개 무료 증정
- **전용 URL**: `/refer/{user_id}` 생성
- **공유 버튼**: 카톡 / 문자 / 링크복사 3종

#### 이벤트 3 — 가족 인증샷 이벤트 (감성 후킹)
- 제목: "부모님 반응샷 공유 이벤트"
- 조건: 제작된 노래 들려드리는 영상/사진 SNS 업로드 (#하이싱 #어버이날선물 태그)
- 리워드: 선정 5명 → 전액 환불 + 내년 어버이날 무료 제작권
